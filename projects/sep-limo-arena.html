System Engineering Project: LIMO Arena Design & Navigation


Project Overview

This project presents a full-stack implementation of a system engineering challenge: designing a realistic, theme-based arena where an AgileX LIMO robot can autonomously navigate, map, and avoid obstacles using ROS 1 and LiDAR-based SLAM. The arena was inspired by Canopy Park at Jewel Changi Airport.

Objectives

  • Apply the SEBoK systems engineering approach
  • Design an arena that balances aesthetics, realism, and navigability
  • Implement autonomous navigation using LiDAR SLAM and the ROS Navigation Stack
  • Conduct validation and verification with the LIMO robot

System Context

The robot system integrates software (ROS 1) and hardware (LIMO with LiDAR). The arena design required careful planning around drive mode (Track Mode), turn radius, sensor range, and platform size. Navigation stacks were tuned using RTABMap and DWAPlanner.

Technical Stack

  • Hardware: AgileX LIMO Robot
  • Sensor: LiDAR, IMU
  • Software: ROS 1 Noetic, RViz, Gazebo, RTABMap
  • Mapping: RTABMap for 3D SLAM
  • Planners: Dijkstra (Global), DWAPlanner (Local)

Role & Contribution

As a Robotics Systems undergraduate, I led arena design and simulation validation, implemented navigation behavior in ROS, and conducted system-level verification and validation under competition criteria.

Verification & Validation

  • Verification: Ensured robot could navigate from start to goal in a controlled environment without collision
  • Validation: Confirmed robot behavior aligned with stakeholder expectations (e.g., team, course instructors)
