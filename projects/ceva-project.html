CEVA Project – Intelligent Pallet & Optimization System


Project Overview

The CEVA Project – Intelligent Pallet & Optimization System aims to develop an autonomous warehouse measurement and optimisation system for palletised cargo. The system captures both weight and dimensional data at a fixed checkpoint using a load cell platform and a vision-based stereo camera system.

Objectives

  • Develop an automated pallet measurement system for logistics optimisation
  • Integrate weight and dimension sensing into a single system
  • Apply ROS 2 for system-level integration and communication
  • Improve efficiency and accuracy of warehouse operations

System Overview

The system combines mechanical structure, load cell sensing, and computer vision into a unified measurement station. ROS 2 is used as the middleware for real-time data communication and integration.

Technical Stack

  • Hardware: Load Cell, HX711, Arduino Uno, ZED2i Camera
  • Software: ROS 2, Python, Visual Studio Code
  • Communication: TCP Bridge + ROS 2 Nodes
  • Vision: Stereo depth-based dimension estimation

Role & Contribution

Responsible for system integration, ROS 2 communication, load cell data handling, testing, and validation of measurement accuracy and synchronization.

Verification & Validation

  • Verification: System correctly captures weight and dimension data
  • Validation: Meets warehouse automation and accuracy requirements
